<p>Firefox team will remove the “disable JavaScript option” in Firefox. Don’t worry if you love this feature for testing. You can still disable the JavaScript through Web-developer plugin.</p>

<p>You can download it from <a href="https://addons.mozilla.org/en-US/firefox/addon/web-developer/" title="https://addons.mozilla.org/en-US/firefox/addon/web-developer/">https://addons.mozilla.org/en-US/firefox/addon/web-developer/</a></p>
