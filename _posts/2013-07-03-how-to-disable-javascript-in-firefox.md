<p>The Firefox team is planning to remove the built-in "Disable JavaScript" option from the browser menu. Don't worry if you rely on this feature for testing — you can still disable JavaScript using the <strong>Web Developer</strong> extension.</p>

<p>You can download it from the Firefox Add-ons page: <a href="https://addons.mozilla.org/en-US/firefox/addon/web-developer/">Web Developer Extension</a></p>
