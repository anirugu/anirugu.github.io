<p>If you want to try a better way to use the select tag in HTML, take a look at the Bootstrap-based Select2 library at <a href="http://ivaynberg.github.io/select2/">http://ivaynberg.github.io/select2/</a></p>

<p>Using this plugin, you can easily search for the option you want to choose.</p>

<p>For example, if you want to search for "xx abc yz", typing "abc" will find it in Select2, whereas native HTML does not support this. Select2 allows you to easily find the option you are looking for.</p>

<p>Many customization options make it awesome.</p>
