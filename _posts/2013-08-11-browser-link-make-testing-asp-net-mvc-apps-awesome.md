<p>A few weeks ago, <a href="http://madskristensen.net/">Mads</a> showed us a cool demo of a new feature in Visual Studio 2013, including Express. One of the coolest features is Browser Link. </p>

<p>Using this feature, you can refresh the browser from Visual Studio. For example, if you have three browsers open, such as Chrome, Firefox, and Opera, you can refresh all of them without going to each browser and clicking the refresh button. This is an awesome feature.</p>

<p>This feature needs JavaScript to be enabled. </p>

<p>For more information watch the videos on Channel9.Msdn.com </p>
