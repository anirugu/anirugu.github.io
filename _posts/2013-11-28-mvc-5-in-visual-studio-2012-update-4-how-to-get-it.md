<p>Even after installing Visual Studio 2012 Update 4, you might notice that there is still no project template for ASP.NET MVC 5. To enable this, you need to install <strong>ASP.NET and Web Tools 2013.1</strong>. For more information, read the official announcement post: <a href="http://blogs.msdn.com/b/webdev/archive/2013/11/18/announcing-release-of-asp-net-and-web-tools-2013-1-for-visual-studio-2012.aspx" title="Announcing release of ASP.NET and Web Tools 2013.1 for Visual Studio 2012">Announcing ASP.NET and Web Tools 2013.1</a>.</p>

<p>You can easily install it using the Web Platform Installer (WPI). The interface will look like this once the installation is complete:</p>

<p><a href="https://gwb.blob.core.windows.net/anirugu/Windows-Live-Writer/Visual-studio-2012_93BB/11_2.png"><img src="/2013_11_28_mvc_5_in_visual_Image1.png" alt="11" title="11" /></a></p>

<p>Now, Visual Studio 2012 (including the Express edition) will display the project template for MVC 5:</p>

<p><a href="https://gwb.blob.core.windows.net/anirugu/Windows-Live-Writer/Visual-studio-2012_93BB/12_2.png"><img src="/2013_11_28_mvc_5_in_visual_Image2.png" alt="12" title="12" /></a></p>
