<p>Recently, I confirmed the MVC support details for Visual Studio 2013 on Channel 9.</p>

<ul>
<li>ASP.NET MVC 5 requires ASP.NET 4. </li>
<li>VS2013 has MVC4 and MVC5 </li>
<li>VS2012 has MVC3 and MVC4 </li>
<li>VS2010 has MVC2 and MVC3  </li>
</ul>

<p>Thanks to <a href="http://hanselman.com/blog">Scott</a> for sharing this information. You can read more details in the comments here: <a href="http://channel9.msdn.com/Events/Build/2013/9-002">http://channel9.msdn.com/Events/Build/2013/9-002</a></p>
